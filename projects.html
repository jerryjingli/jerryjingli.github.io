Knowledge Diffusion in Stack Overflow

Hyperlinks are essential knowledge-oriented entities in Stack Overflow. Here we are concerned with the hyperlinks that the users diffuse in their questions and answers. What types of hyperlinks are most frequently diffused? How has a particular hyperlink been diffused over time? Which web sites or which users influence the diffusion of a particular hyperlink, and how? To address these questions, we propose a graph-based knowledge cascade model that combines temporal-spatial perspectives of hyperlinks diffusion in a shared context. Based on this model, we identify temporal patterns of hyperlink diffusion, and infer three kinds of graphs, i.e., knowledge influence graph, knowledge disseminator graph, and user interaction graph to study influencing web sites, key disseminators, and active users in hyperlink diffusion, as well as scale-free property and small-world effect of the inferred graphs. In addition to the large-scale empirical study, we present the future directions and tools that can leverage a better understanding of knowledge diffusion to enhance user experience in using Q&A services in Stack Overflow.
http://knowledge-diffusion.appspot.com/.

scvRipper: A Tool for scraping time-series interaction data from screen-captured videos

scvRipper is a computer-vision-based video scraping technique to recognize window-based applications in screen-captured videos, and to extract application-specific content from the recognized application windows (e.g., code fragments in file editor, error messages in IDE console output, URLs in URL bar of web browser, and keywords in search box of search engine). scvRipper tool takes as input a screen-captured video, i.e., a time-series stream of screenshots taken by screen-capture tools such as Snagit. It produces as output a time-series interaction data (i.e., software usage and application content) reverse-engineered from the video.
